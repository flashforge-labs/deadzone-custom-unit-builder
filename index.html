Guide / FAQ

What This Is

This is an unofficial fan-made custom-unit points estimator. It uses derived private calibration weights, not official profiles, and is in no way affiliated with Mantic Games.

Using The Builder

Build one unit profile, add it to the army list, then repeat. The total at the top uses unit cost multiplied by quantity.

Saving Lists

Save Template downloads a JSON file that can be loaded again later. Print / PDF opens the browser print dialog.

Special Rules

Special rules are intended for Leaders and Legends. For other roles they stay visible but do not affect the estimate.

How Accurate Is It?

The model reconstructs known unit profiles with roughly 98-99% practical accuracy, with almost all known units landing within 1 point. For new custom units, treat the result as a strong starting estimate, not a substitute for playtesting.

Balance

The app is designed to help make balanced units, but it is intended for fun and casual games only. In competitive games it can be abused with weird scoring at high point values, or with broken combinations that do not appear in real lists. Making custom units should be a fun process, but it needs restraint and common sense to avoid breaking the game.
